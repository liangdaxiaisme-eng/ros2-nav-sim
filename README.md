# 🤖 ROS2 Autonomous Navigation Simulator

**赛博朋克风格**的移动机器人自主导航仿真系统，模拟 ROS 2 Nav2 全套功能，纯 Python 实现，无需安装 ROS。

![Lines](https://img.shields.io/badge/code-3355%20lines-brightgreen)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

### 🗺️ 地图与导航
- **复杂地图生成** — 200×200 栅格，会议室/办公室/仓库/实验室/走廊/大厅
- **A\* 全局路径规划** — 8-connected, Octile heuristic, 代价地图膨胀层
- **DWA 局部路径规划** — 动态窗口法实时避障
- **航点导航状态机** — IDLE → PLANNING → NAVIGATING → ARRIVED
- **卡住检测与自动重规划**

### 🤖 机器人仿真
- **差速驱动运动学模型** — 含高斯噪声
- **360° 激光雷达** — 180束, 8m 量程, 射线投射
- **IMU 模拟** — 四元数姿态 + 角速度 + 线加速度
- **里程计模拟** — 含漂移噪声
- **碰撞检测**

### 🎨 视觉特效 (赛博朋克)
- **CRT 扫描线** + **暗角效果**
- **霓虹发光管线** — 路径/边框/状态指示
- **粒子系统** — 轨迹尾焰、到达烟花、卡住火花
- **雷达扫描波** — 左上角实时旋转
- **速度波形图** — 线速度/角速度实时曲线
- **屏幕震动** — 到达/碰撞/卡住触发
- **随机故障特效** — 随机水平条纹偏移
- **网格叠加层**
- **启动动画序列** — 仿真系统自检日志

### 📊 HUD 遥测面板
- 实时位姿/速度/导航统计
- 动画速度条 (带辉光)
- 波形图 (线速度/角速度)
- 滚动日志
- 小地图
- FPS / 帧计数器

## 📦 安装

```bash
cd ros2-nav-sim
pip install pygame numpy scipy
```

## 🚀 运行

### GUI 模式 (推荐)

```bash
python3 main.py
```

启动后会先播放系统自检动画，然后自动开始导航。

### 无头模式 (生成视频)

```bash
# 需要虚拟显示器
Xvfb :99 -screen 0 1920x1080x24 -ac &
export DISPLAY=:99

# 参数: 输出路径 [秒数]
python3 generate_video.py demo.mp4 30
```

### 仅测试地图

```bash
python3 maps/complex_map.py
```

## 🎮 操作指南

| 按键 | 功能 |
|------|------|
| `1` - `9` | 切换航点预设路线 (共9条) |
| `Space` | 暂停 / 继续 |
| `R` | 重置机器人到起点 |
| `C` | 切换代价地图叠加 |
| `L` | 切换激光雷达射线 |
| `T` | 切换历史轨迹 |
| `+` / `-` | 缩放视角 |
| 鼠标拖拽 | 平移视角 |
| `Q` / `Esc` | 退出 |
